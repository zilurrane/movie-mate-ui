# MovieMateUI
 UI repository for MovieMate project

### Demo

LINK: https://movie-mate-ui.herokuapp.com/

### Getting started

  1. Clone
  ```
   git clone https://github.com/zilurrane/movie-mate-ui.git
  ```

  2. Install packages
  ```
   npm install
  ```
  
  3. Build
  ```
   npm run build
  ```
  
  4. Start
  ```
   npm start
  ```
